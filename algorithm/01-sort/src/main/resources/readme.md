### 常数时间操作是什么？
如果一个操作的执行时间不以具体样本量为转移，每次执行时间都是固定时间。
称这样的操作为常数时间的操作
常见的常数时间操作
1.常见的算术操作
2.常见的位运算（>>,>>>,<<,|,&,^等）
. >>：带符号位移，如果该数为正,则高位补0,若为负数,则高位补1
. >>>：不带符号位移
3. 赋值、比较、自增、自减操作
4. 数组寻址操作

执行时间固定的操作都是常数时间的操作
执行时间不固定的操作，都不是常数时间的操作
